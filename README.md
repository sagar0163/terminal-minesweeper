# Terminal Minesweeper

Classic Minesweeper game for your terminal.

## Features

- 💣 Classic Minesweeper gameplay
- 🎯 Cursor-based controls
- 🚩 Flagging system
- 3 difficulty levels

## Requirements

```bash
pip install keyboard
```

## Run

```bash
python minesweeper.py
```

## Controls

- Arrow keys: Move cursor
- Enter: Reveal cell
- F: Flag/Unflag
- Q: Quit
