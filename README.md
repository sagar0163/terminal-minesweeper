# Terminal Minesweeper 🎯

Classic Minesweeper game for Linux terminal - Enhanced Edition v2.0!

## Version 2.0 - Enhanced

### New Features in v2.0

| Feature | Description |
|---------|-------------|
| **First Click Safe** | Never hit a mine on first click |
| **Chord clicking** | Click numbers to open neighbors |
| **Question marks** | Mark uncertain cells with ? |
| **Move counter** | Track your moves |
| **Timer** | Track completion time |
| **Wrong flags** | Shows ❌ for incorrectly flagged cells |
| **Cursor highlight** | See your current position |
| **Color numbers** | Numbered cells are color-coded |

### Controls

| Key | Action |
|-----|--------|
| Arrow Keys | Move cursor |
| Enter | Reveal cell |
| F | Toggle flag |
| ? | Toggle question mark |
| Q | Quit |

### How to Play

1. **Reveal cells** - Click to reveal a cell
2. **Numbers** - Tell you how many mines are adjacent
3. **Flags** - Mark suspected mines with F
4. **Win** - Reveal all non-mine cells
5. **Lose** - Hit a mine 💥

### Difficulty Levels

| Level | Size | Mines |
|-------|------|-------|
| Easy | 9×9 | 10 |
| Medium | 16×16 | 40 |
| Hard | 30×16 | 99 |

### Tips

- Numbers show adjacent mine count
- Empty cells auto-reveal (flood fill)
- First click is always safe!
- Chord: click a number to open all neighbors if flags match the number
